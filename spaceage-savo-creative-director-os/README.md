# Space Age Visual Orchestrator — Creative Director OS v1.0

Space Age Visual Orchestrator (SAVO) is a pre-design AI Creative Director Operating System for Space Age AI Solutions.

SAVO does **not** start by building websites. It starts by diagnosing the business, brand, market, audience, trust requirement, conversion goal, visual opportunity, storytelling route, motion tolerance, and proof strategy. Only after that does it create a Figma/Stitch blueprint and universal build handoff.

## What SAVO Decides

SAVO answers this question before design or code:

> How should this business capture attention, tell its story, build trust, and generate conversions?

Then it selects the right visual system, layout strategy, motion choreography, material language, proof architecture, and CTA path.

## Locked Defaults

These are always present unless explicitly overridden:

- Jumbo typography
- Fullscreen hero sections
- Video-first hero sections
- Premium spacing
- Strong visual hierarchy
- Cinematic composition
- Memorable first impression

## Adaptive Systems

These are selected only when strategically justified:

- GSAP ScrollTrigger
- Pinned storytelling
- Horizontal scrolling
- FLIP layout transitions
- Layered parallax
- Glassmorphism
- Liquid glass approximation
- Skeuomorphism
- Image sequences
- Scroll scrubbing
- Dashboard expansions
- Grid to feature transformations
- Card to fullscreen transformations

## What Makes This Different

Most AI site builders ask: "What effects should I use?"

SAVO asks:

- What must the visitor believe first?
- What emotion should the first screen create?
- What proof must appear before the CTA?
- What level of visual risk helps conversion instead of hurting it?
- What motion guides attention instead of showing off?
- What layout pattern makes the offer easier to understand?
- What build path can a real model execute without drifting?
