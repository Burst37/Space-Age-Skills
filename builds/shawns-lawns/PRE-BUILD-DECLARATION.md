## 12. Why I Chose This

Shawn's Lawns has zero web presence (Bucket 1). The conversion path is simple: see the transformation, trust the quality, call for a quote. Every decision here serves that path.

- **Mode A video (transformation):** Nothing sells lawn service like seeing the before/after. It's the product.
- **5 modules only:** A landscaper doesn't need shaders or WebGL. Clean, fast, mobile-first wins.
- **Dark premium aesthetic:** Elevates the brand above every other local lawn company in Mesquite. Shawn looks like a professional operation, not a handyman with a truck.
- **Single-file HTML:** Fast to build, fast to load, easy to hand off. No framework overhead.
- **Claude Code as builder:** Right complexity level. Parallel agents are for batch mode.
- **Gemini 3.1 Flash TTS widget:** Differentiates immediately. Visitors can talk to "Shawn" 24/7. Handles quote inquiries, service questions, booking intent — without Shawn picking up the phone.
