## VL-01 DARK GLASSMORPHISM SYSTEM

The Space Age default design system. Apply to every project unless brand overrides.

### Core Color System

```css
:root {
  /* Surfaces */
  --surface-base:        #050508;
  --surface-raised:      #08080C;
  --surface-overlay:     #0D0D14;
  --surface-glass:       rgba(255, 255, 255, 0.06);
  --surface-glass-hover: rgba(255, 255, 255, 0.10);
  
  /* Primary (Electric Blue default — override per brand) */
  --color-primary:       #2979FF;
  --color-primary-glow:  rgba(41, 121, 255, 0.20);
  --color-primary-dark:  #1A56D6;
  
  /* Text */
  --text-primary:        #FFFFFF;
  --text-secondary:      #A0A0B0;
  --text-tertiary:       #606070;
  
  /* Borders */
  --border-subtle:       rgba(255, 255, 255, 0.06);
  --border-default:      rgba(255, 255, 255, 0.10);
  --border-strong:       rgba(255, 255, 255, 0.18);
}
```

### Industry Color Overrides

Swap `--color-primary` based on client industry:

```yaml
INDUSTRY_COLORS:
  tech_ai:      "#2979FF"  # Electric blue (default)
  health:       "#00C853"  # Medical green
  finance:      "#FFD600"  # Premium gold

  music:        "#AA00FF"  # Creative violet

```

### Typography Stack


```css

```

### Glassmorphism Components

```css
/* Glass card */
.glass-card {
  background: var(--surface-glass);
  border: 1px solid var(--border-default);
  border-radius: 20px;
  backdrop-filter: blur(20px) saturate(150%);
  -webkit-backdrop-filter: blur(20px) saturate(150%);
  box-shadow: 0 24px 48px rgba(0, 0, 0, 0.4);
}

/* Primary button */
.btn-primary {
  background: var(--color-primary);
  border-radius: 10px;
  padding: 14px 32px;
  font-family: var(--font-heading);
  font-size: 0.875rem;
  font-weight: 900;
  letter-spacing: 0.08em;
  text-transform: uppercase;
  color: #FFFFFF;
  border: none;
  cursor: pointer;
  transition: all 200ms ease;
}
.btn-primary:hover {
  filter: brightness(1.15);
  box-shadow: 0 0 32px var(--color-primary-glow);
}
```

## DESIGN.MD OUTPUT FORMAT

```yaml
# DESIGN.md
# Generated by sa-design-md skill
# Space Age AI Solutions — VL-01 Design System

project:
  name: "[Brand Name]"
  date: "[Date]"
  version: "1.0.0"

design_system: "VL-01 Dark Glassmorphism"

colors:
  surface_base: "#050508"
  surface_raised: "#08080C"
  primary: "[Industry color or brand color]"
  primary_glow: "rgba([r], [g], [b], 0.20)"
  text_primary: "#FFFFFF"
  text_secondary: "#A0A0B0"
  border_default: "rgba(255, 255, 255, 0.10)"

typography:
  heading: "Orbitron 900"
  body: "DM Sans"
  data: "JetBrains Mono"
  cdn: "fontsource"

motion:
  transition_normal: "320ms cubic-bezier(0.16, 1, 0.3, 1)"
  transition_slow: "560ms cubic-bezier(0.16, 1, 0.3, 1)"
  transition_cinematic: "900ms cubic-bezier(0.25, 1, 0.5, 1)"

components:
  glass_card:
    background: "rgba(255, 255, 255, 0.06)"
    border: "1px solid rgba(255, 255, 255, 0.10)"
    border_radius: "20px"
    backdrop_filter: "blur(20px) saturate(150%)"

downstream:
  - ui-ux-designer
  - cinematic-website-builder
  - SA-immersive-reveal
  - SA-3d-slider
  - framer-shaders
```

## WHAT TO AVOID

- Don't override VL-01 core without brand justification
- Don't use Google Fonts — always Fontsource CDN
- Don't skip the DESIGN.md file before starting any site build
