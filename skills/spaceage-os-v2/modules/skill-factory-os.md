# Space Age Skill Factory OS

## Mission

Turn raw frameworks into elite reusable Space Age skills and operating systems.

## Build, Don't Talk Protocol

When the user says "build," "create," "redo," "save," or uploads a repo/file:

1. Inspect the source
2. Extract the highest-value architecture
3. Remove bloat and weak assumptions
4. Add Space Age commercial workflows
5. Save the rebuilt skill file or package
6. Return download links

## Reverse Engineering Checklist

```yaml
source_analysis:
  identity: [name, purpose, intended agent harness, install method]
  capabilities: [commands, workflows, file conventions, trigger phrases, reusable patterns]
  quality: [strengths, weaknesses, missing safeguards, missing output contracts, missing verification]
  upgrade_targets: [agency usefulness, coding reliability, design taste, automation potential, knowledge management, monetization potential]
```

## Required Skill Anatomy

Every rebuilt skill must contain:

```yaml
frontmatter:
  name: lowercase-hyphen-name
  description: trigger-rich description
  version: semantic version
  source_inspiration: list
sections:
  - ROLE
  - WHEN TO USE
  - WHEN NOT TO USE
  - INPUT CONTRACT
  - OPERATING WORKFLOW
  - OUTPUT CONTRACT
  - QUALITY GATES
  - SPACE AGE EXTENSIONS
  - EXAMPLES
  - FAILURE MODES
```

## Upgrade Layers

```yaml
layers:
  orchestration: [trigger routing, handoff rules, dependency order]
  research: [Firecrawl-compatible web extraction, competitor intelligence, source grounding]
  design: [Impeccable anti-slop standards, Space Age moodboard routing, VL-01 visual rules]
  engineering: [Superpowers planning loop, Karpathy verification loop, test-first when code changes]
  knowledge: [Obsidian frontmatter, backlinks, vault-ready naming]
  packaging: [one combined markdown, one installable skill folder, one zip package]
```

## File Naming Rules

```yaml
folders: lowercase-hyphen
skill_file: SKILL.md
combined_master: SpaceAge_[SystemName]_vX.md
zip: SpaceAge_[SystemName]_vX.zip
no_prefix_in_skill_name: true
```

## Done Means Saved

A skill-factory task is not complete until an artifact exists in the target workspace.
