## CONTENT TYPE SYSTEM

### TYPE A — Educational Carousel
**Purpose:** Teach something valuable, build authority, get saves
**Slides:** 6-10
**Structure:**
```
Slide 1 (Cover):    Hook question or bold statement + character facing OUT
Slide 2:            The tension / problem / "most people do X wrong"
Slide 3:            The turn / your perspective
Slides 4-7:         The content (1 tip/lesson per slide)
Slide 8-9:          Summary or bonus tip
Last Slide (CTA):   Follow/save/comment CTA + character pointing at action
```
**Typography:** TT-02 (Mixed Weight) + TT-03 (Chromatic Split) for hook word
**Best styles:** Moodboard R · F · T · O

### TYPE B — Product/Service Post
**Purpose:** Showcase a product, drive purchase, announce
**Format:** Single post (1:1 or 4:5)
**Structure:**
```
Hero visual:    Product/service front and center
Headline:       Bold, benefit-focused (not feature)
Subline:        One line price or CTA
Brand mark:     Top corner, subtle
```
**Typography:** TT-04 (Condensed Full-Width) or TT-01 (BG Bleed) behind product
**Best styles:** Moodboard Q · P · S · A

### TYPE C — Authority/Thought Leadership Post
**Purpose:** Position as expert, build trust, create conversation
**Format:** Single post or 2-3 slide mini-carousel
**Structure:**
```
Strong statue/figure OR portrait of subject
Bold quote or statement as headline
Accent word highlighted (TT-03)
Minimal body copy — let the visual carry weight
```
**Typography:** TT-03 + TT-04
**Best styles:** Moodboard T · J · H

### TYPE D — Brand Awareness Post
**Purpose:** Remind audience of brand, stay top-of-mind
**Format:** Single post, high visual impact
**Structure:**
```
Brand personality expressed visually
Minimal text — logo + tagline max
Color system doing the heavy lifting
```
**Typography:** TT-04 or none
**Best styles:** Moodboard A · D · K · S

### TYPE E — Promotional/Offer Post
**Purpose:** Drive direct action — click, buy, sign up
**Format:** Single post or 2-slide reveal
**Structure:**
```
Slide 1: Big hook — the result or the offer headline
Slide 2 (optional): The details, price, deadline
Always: Clear CTA button or arrow graphic
Always: Urgency element (limited, today only, etc.)
```
**Typography:** TT-05 (Neon Glow) for gaming/youth · TT-04 for authority
**Best styles:** Moodboard P · F · M · A

### TYPE F — Behind The Scenes / Personal Brand
**Purpose:** Build connection, humanize brand, show process
**Format:** Single post, documentary feel
**Structure:**
```
Real or AI-generated workspace/action visual
Honest, conversational headline
Raw typography — not over-designed
Brand colors present but understated
```
**Typography:** Clean body type, minimal effects
**Best styles:** Moodboard H · L · G

### TYPE G — Results/Proof Post
**Purpose:** Social proof, testimonials, case study
**Format:** 3-5 slide carousel
**Structure:**
```
Slide 1: The result number — BIG — with context
Slide 2: Before state (the problem)
Slide 3: After state (the result)
Slide 4: Quote from client or key stat
Slide 5: CTA — "You're next"
```
**Typography:** TT-01 for the big number · TT-02 for context
**Best styles:** Moodboard A · L · T

### TYPE H — Product Reveal / Launch
**Purpose:** Launch a new product, create excitement
**Format:** 3-5 slides or single cinematic post
**Structure:**
```
Teaser slide (optional): Blurred or partial product
Reveal slide: Product hero shot, cinematic bg
Feature slides: 1-3 key features, each with visual
CTA: Price + buy link
```
**Typography:** TT-04 for product name · TT-08 for spec callouts
**Best styles:** Moodboard Q · P · S · O

## CHARACTER / MASCOT SYSTEM

Characters anchor social content. Every high-performing carousel uses one.

### Option 1 — AI-Generated Character (NanoBanana/Midjourney)
Generate a consistent 3D character using the cinematic-prompt-director skill.
Lock the character with a **Consistency Core** clause on every regeneration.
Output: 8-12 poses needed minimum:
```
Pose Set A (Required):
- Facing camera directly (cover slides, hook)
- Profile/side view left (content slides)
- Profile/side view right (content slides)
- Back to camera, looking at content (narrative slides)
- Pointing RIGHT (for CTA pointing at text)
- Pointing DOWN (for CTA pointing at button)
- Arms crossed / authority pose
- Surprise/reaction expression

Pose Set B (Nice to have):
- Sitting/relaxed (tips, listicles)
- Looking up (aspirational content)
- Facepalm/frustration (pain point slides)
- Celebration/fist pump (results slides)
```

### Option 2 — Client's Existing Brand Character
Extract from brand kit, place on transparent background.
Always: desaturate slightly and add shadow to ground on slide.

### Option 3 — No Character (Type-Forward Approach)
For: Premium/luxury brands, B2B, high-fashion, financial services
Replace character with: Strong typography, product, or abstract 3D object

### Character Placement Rules
```
Cover Slide:        Character 40-60% of frame height, centered or left third
Content Slides:     Character 30-40% of frame, LEFT side, content on RIGHT
CTA Slide:          Character pointing/gesturing toward the CTA button
Never:              Character behind too much text — one or the other leads
Always:             Character shadow/ground shadow for depth
Always:             Character facing INTO the slide (not off-frame)
```

```
STEP 1: Choose Content Type (A through H)
STEP 2: Choose Moodboard (A through T)

STEP 6: Apply typography system
STEP 7: Quality check (see checklist below)
```

## QUALITY CHECKLIST (Before Every Delivery)

### Visual
- [ ] Is there ONE clear focal point? (Character, product, or headline — not all three fighting)
- [ ] Does it stop the scroll in 0.3 seconds?
- [ ] Is the hierarchy obvious? (What to read first, second, third)
- [ ] Does the color system match the brand?
- [ ] Is text legible on mobile at thumbnail size?
- [ ] Does it have depth? (Foreground/midground/background layers)
- [ ] Is there a shadow/ground element so the character isn't floating?

### Typography
- [ ] One display font only (unless TT-02 intentional mix)
- [ ] ONE accent-colored word — not five
- [ ] Body text minimum 24px for social (reads on mobile)
- [ ] No widows (single word on last line of body text)
- [ ] Headline and body NOT the same weight/size

### Carousel-Specific
- [ ] Slide counter visible on every slide?
- [ ] Brand handle consistent position on every slide?
- [ ] Character facing correctly for each slide type?
- [ ] Can each slide stand alone AND work as part of the series?
- [ ] Does Slide 1 make someone want to swipe?
- [ ] Does the last slide have a clear CTA?

### Technical
- [ ] Exported at correct resolution (1080px+ on short side)
- [ ] PNG format for graphics (not JPG which compresses text poorly)
- [ ] JPG for photo-dominant posts (smaller file size, equally sharp)
- [ ] Canva template saved and labeled
- [ ] All assets in client folder

## COMPETITOR INTELLIGENCE

**What separates premium social media design:**
1. **Character consistency** — same character across all content creates brand recognition
2. **System thinking** — templates that scale, not one-offs
3. **Cinematic backgrounds** — not stock photos, AI-generated environments
4. **Typography discipline** — ONE treatment per post, executed perfectly
5. **3D element integration** — renders, objects, cutouts that feel editorial
6. **Color grading** — every post feels like it came from the same film set
7. **Text hierarchy mastery** — they know which word gets huge, which gets small
